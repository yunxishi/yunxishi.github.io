1.变量的命名和使用

1.口变量名只能包含字母、数字和下划线。变量名能以字母或下划线打头，但不能以数字打头。例如，可将变量命名为 message_1，但不能将其命名为1_message。

2.变量名不能包含空格，但能使用下划线来分隔其中的单词。例如，变量名 greeting_message 可行，但变量名 greeting message会引发错误。

3.不要将Python关键字和函数名用作变量名，即不要使用Python保留用于特殊用途的单词，如print

4.变量名应既简短又具有描述性。例如，name比n好，student_name比s_n好，name_length比length_of_persons_name好。

5.慎用小写字母1和大写字母O，因为它们可能被人错看成数字1和0

if语句

简单的if语句

第一行可以包含任何条件测试，而在紧跟在测试后面的缩进代码块中，可以执行任何操作。如果条件测试的结果为True，python就会执行紧跟在if语句后面的代码，否则会忽略

if—else语句

在if—else语句中，如果if测试结果为true，则执行if的print，如果if测试结果为false，则执行else的print

if—elif—else

python只执行if—elif—else结构中的一个代码块，它一次检查每个条件测试，直到条件测试通过

使用多个elif代码

可以添加多个elif来增加多个条件

在if—elif后面不一定需要else代码块，这个时候可以省略else

总结

如果只执行一个代码块，就if—elif—else结构，如果执行多个代码块，就是用一系列独立的if语句
